# [[marginal]] [[cost]]

- MC
- additional costs resulting from additional output
- change in [[total cost]] / change in quantity
- curve is an upside-down [[marginal product|MP]] curve
- in the [[short-run]], [[supply]] curve is MC curve when the MC is above [[average variable cost|AVC]] in a [[perfect competition|perfectly competitive market]]
