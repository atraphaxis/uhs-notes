# B. F. Skinner

- 1904-1990
- supported [[behaviorism]]
- rejected introspection and, with it, [[structuralism]]
- studied how consequences affect behavior
	- [[operant conditioning]]
- used operant conditioning to explain how humans learn [[language]]
	- association
	- imitation
	- reinforcement
